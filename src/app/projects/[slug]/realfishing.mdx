REAL FISHING is a 2D networked platformer about real fishing. It was made over a
2-day game jam that was originally VERY SCARY themed but then turned into
fishing, sort of. We ended up making a side scrolling hook swinging game about
real fishing.

- Made by: Max Chung, Winston Liang, Julia Ahopelto
- Made with: Lua, Defold
- Repository: [https://github.com/maxrchung/real-fishing](https://github.com/maxrchung/real-fishing)
- Play: [https://github.com/maxrchung/REAL-FISHING/releases/tag/1.0.0](https://github.com/maxrchung/REAL-FISHING/releases/tag/1.0.0)
