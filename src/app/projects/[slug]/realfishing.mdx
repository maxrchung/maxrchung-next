REAL FISHING is a 2D networked platformer about real fishing. It was made over a
weekend game jam. The theme was originally "VERY SCARY" but then turned into
fishing, sort of. We ended up making a side scrolling hook swinging game about
real fishing.

- Made by: Max Chung, Winston Liang, Julia Ahopelto
- Made with: Lua, Defold
- Repository: [REALFISHING](https://github.com/maxrchung/REALFISHING)
- Download (Windows): [https://github.com/maxrchung/REALFISHING/releases](https://github.com/maxrchung/REALFISHING/releases)
