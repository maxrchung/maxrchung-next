Ride in the Rain is a 3D networked racing game made over a 3 day game jam using Godot. Race alone or co-op with others and try to get the fastest time.

- Made with: Godot
- Made by: Max Chung, Winston Liang, Julia Ahopelto
- Download (Windows): [Final final game jam build](https://github.com/maxrchung/ride_in_the_rain/releases/tag/v0.0) (186MB)
